wirelessproblem

发表于 2019-08-21 分类于 other

问题描述

打开电脑发现无法联无线网，使用以下命令检查：

  1. 发现两个接口：eno1和lo,并没有发现无线网卡
  2. 退出Ubuntu系统，进入win10依旧未发现无线网卡，怀疑无线网卡损坏。
  3. 外出购买内置无线网卡rtl8723be
    • 安装后win10系统默认关闭wifi，睡眠后开启wifi接口可打开，重启后又关闭。尝试用笔记本默认硬件开关F12和fn+ F12，只能切换飞行模式和wifi，但是wifi仍处于关闭状态
    • Ubuntu系统处于飞行模式并且显示wifi硬件开关关闭。尝试笔记本默认wifi硬件开关，按键无效或终端输入~符号。
  4. ubuntu系统下尝试下列命令解决问题 显示 soft blocked no
    hard blocked yes
    说明硬件开关阻塞 未造成任何改变，此方法仅适合与开关软件阻塞
    尝试挂起系统解决问题 找不到pm命令，连接网线，通过有线网络下载pm 断开有线网络 回车唤起系统，发现wifi正常使用。
    但是每次开机后挂起很麻烦，所以寻找其他解决办法 尝试开闭模块重新激活，失败 失败，重启后问题仍无法解决 原来可以链接的wifi显示连接失败，需要激活
    重启后问题未得到解决 卸载后重启可以通过挂起联网
