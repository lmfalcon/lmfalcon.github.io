# Next主题配置

## tags配置

1. 打开主题配置文件
2. /menu	搜索菜单配置信息，将tags的注释
3. sudo hexo new page "tag" 创建新页面
4. 在source下出现新的文件夹tags,编辑里面的index.md文件
5. 添加type：tags

## search配置

1. 本地安装npm install hexo-generator-search
2. 修改主题配置文件
3. /local search	找到相关位置将enable:true设置
