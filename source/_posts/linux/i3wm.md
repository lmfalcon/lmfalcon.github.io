# i3的基本使用

- i3的配置文件在~/.config/i3/config 中记不得使用方式可以直接查询
- mod键默认设置为super键
- mod + Enter 打开命令行
- mod + Shift + q 退出窗口
- mod + 数字 切换/不存在时生成 不同窗口
- mod + Shift + 数字 切换程序到不同的窗口
- mod + r 调整窗口的大小
- mod + hjkl 调整需使用窗口，也可以通过方向键调整
- mod + d 小命令行，不显示运行结果但是可以执行命令，还可以提示当前系统有什么命令可以执行
